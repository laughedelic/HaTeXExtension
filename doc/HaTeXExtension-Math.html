Description

This module provides those TeX commands, that are usually used in math mode. Also it makes LaTeX type an instance of Num and Frac classes and derives Eq instance.

Application operator

(.:) :: LaTeX -> LaTeX -> LaTeX

Application operator - encloses second arg in parenthesis. Example: f.:(x + y) means f \left( x + y \right) in TeX.
